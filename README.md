# 🛒 MERN E-Commerce

Full-stack e-commerce app — Vite + React + Redux Toolkit (Frontend) | Node.js + Express ESM + MongoDB (Backend)

## Structure

```
mern-ecommerce/
├── frontend/   → Vite + React + Redux Toolkit + Tailwind
└── backend/    → Node.js ESM + Express + MongoDB + Mongoose
```

## Quick Start

```bash
# Backend
cd backend && cp .env.example .env  # fill in your values
npm install && npm run dev

# Frontend (new terminal)
cd frontend && npm install && npm run dev
```

## Demo Accounts (after seeding)

- Admin : admin@demo.com / admin123
- User : user@demo.com / user123

```bash
# Seed demo data
cd backend && npm run seed
```
